# SkillXchange 🔄
 
A skill exchange platform where users can teach and learn from each other through real-time chat, sessions, and workshops.
 
## ✨ Features
 
- **User Profiles** - Register and showcase your skills
- **Skill Matching** - Find partners for skill exchange
- **Real-time Chat** - Message with file sharing
- **Video Sessions** - Schedule learning meetings with Google Meet
- **Workshops** - Join group learning activities
- **Feedback System** - Rate and review partners
 
## 🛠️ Tech Stack
 
**Frontend:** React, Vite, Tailwind CSS, Socket.io  
**Backend:** Node.js, Express, MongoDB, Socket.io  
**sessision:** Google Meet integration
